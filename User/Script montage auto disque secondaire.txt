Parfait ! 😎 On peut améliorer le script pour qu’il utilise le label du disque, si celui-ci existe, comme nom de point de montage, plutôt que /run/media/<user>/sdb1.

Ainsi, le montage sera plus lisible et intuitif.

1️⃣ Script bash amélioré – montage avec label si disponible
#!/usr/bin/env bash
# ==============================================
# mount-extra-disks.sh
# Monte tous les disques internes secondaires libres
# Utilise le label si disponible, sinon le nom de partition
# Accessible à l'utilisateur courant
# ==============================================

username=$(whoami)

# Détecte le disque système pour l'ignorer
system_disk=$(lsblk -no NAME / | head -n1)

# Boucle sur tous les disques internes secondaires (SATA et NVMe)
for dev in $(lsblk -ndo NAME,TYPE | awk '$2=="disk"{print $1}' | grep -v "^${system_disk}"); do
    # Boucle sur les partitions du disque
    for part in $(lsblk -ln /dev/$dev | awk '$6=="part"{print $1}'); do
        device="/dev/$part"
        # Vérifie que la partition n'est pas déjà montée
        if ! mount | grep -q "$device"; then
            # Essaie de récupérer le label du disque
            label=$(lsblk -no LABEL "$device")
            if [ -n "$label" ]; then
                mount_point="/run/media/${username}/${label}"
            else
                mount_point="/run/media/${username}/$part"
            fi
            mkdir -p "$mount_point"
            mount -o users,nofail,noatime "$device" "$mount_point"
            chown -R "$username":users "$mount_point"
            echo "Monté $device → $mount_point"
        fi
    done
done
✅ Points clés :

lsblk -no LABEL "$device" → récupère le label de la partition.

Si le label existe → point de montage /run/media/<user>/<label>

Sinon → fallback /run/media/<user>/sdb1 ou équivalent

Toutes les partitions secondaires libres sont montées automatiquement

Permissions pour l’utilisateur courant sont appliquées (chown -R $username:users)

2️⃣ Service systemd user (inchangé)
[Unit]
Description=Monte tous les disques internes secondaires libres pour l'utilisateur
After=default.target

[Service]
Type=oneshot
ExecStart=%h/bin/mount-extra-disks.sh
RemainAfterExit=true

[Install]
WantedBy=default.target

%h → home directory de l’utilisateur

Activation :

systemctl --user daemon-reload
systemctl --user enable --now mount-extra-disks.service
3️⃣ Résultat

Tous les disques internes secondaires sont montés automatiquement.

Les points de montage sont lisibles grâce au label des disques.

Compatible NixOS, Fedora, Ubuntu, Debian, Arch…

Service préservé lors des mises à jour, car user-level.
